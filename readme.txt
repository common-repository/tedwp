A simple Classic Block Editor plugin for writers and bloggers. Alternative to Gutenberg.

== Description ==
TedWP is an intuitive and easy-to-use WordPress editor plugin that lets you focus on writing bold and clear blog articles.

### Easy to write blog posts
With TED you can easily write, and edit blog content and format and publish blog posts quickly. Saves your time and energy.

We believe that TedWP could be a useful tool for people looking for an alternative to the Classic Editor plugin or the Gutenberg editor. 

### Frontend buttons
It adds a Create New Post and Edit Post buttons on the frontend for your blog to quickly access the editor. So people who need a WP frontend post editor can also make use of this editor plugin. 


## Content Editor for WordPress
It is a visual WYSIWYG content editor that comes with options similar to the TinyMCE editor but with a block interface. 


### Basic blocks
Can add content as blocks and comes with basic block types such as Text, Headings, Quotes, Images, Lists, Tables, and more.

### Clean and intuitive interface
It’s a fast-loading editor that comes with a clean interface that lets you focus on writing. 

No barriers between the ideas in your head and writing them on your blog. So you can just focus on writing.

### Better mobile writing experience
Ideas don’t wait until you get to a PC.
With TED editor you can write on any device such as mobile phones and tablets, regardless of screen size.

### Frequently updated
You can expect new features, bug fixes, and security updates regularly.

== Installation ==
1. Upload the plugin to your website.
2. Activate it.
3. Go to any page or post
4. Click Ted Editor Button
5. Edit Text and Save
